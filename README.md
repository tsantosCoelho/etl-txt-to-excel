# 📊 TXT to Excel ETL

ETL automation in Python to structure accounting data from TXT files and generate an Excel output.

## 🚀 Overview

Processing unstructured accounting records and transforming them into a structured dataset for analysis.

## ⚙️ Features

* TXT file parsing
* Extraction of dates, histories, and values
* Data cleaning and conversion
* Export to Excel

## 🛠️ Technologies

Python • Pandas • Regex • OpenPyXL

## 🎯 Objective

Demonstrate a basic ETL pipeline applied to accounting data.
